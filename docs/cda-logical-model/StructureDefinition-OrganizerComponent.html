Logical Model: OrganizerComponent (CDA Class) ( Abstract )

Official URL: http://hl7.org/cda/stds/core/StructureDefinition/OrganizerComponent Version: 2.1.0-draft2-mb
Active as of 2023-10-04 Computable Name: OrganizerComponent

CDA entries represent the structured computer-processable components within a document section. Each section can contain zero to many entries.
