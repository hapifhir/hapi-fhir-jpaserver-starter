Welcome to the AU Core IG, Customized HAPI FHIR Public Test Server (R4 FHIR) provided by Whitefox Cloud Consulting. This server is specifically tailored to the Australian Core Implementation Guide (AU Core IG) and is accessible using the FHIR REST API or HAPI client implementation.

Important Notice: This is a testing environment and not a production server. Do not store any sensitive or confidential information here. The server is regularly purged and reloaded with predefined test data to ensure a consistent testing environment. Please note that preloaded test data cannot be deleted.

About This Server: This server offers a full implementation of the FHIR Specification (AU Core IG) using a completely open-source software stack. It is constructed from various modules of the HAPI FHIR project, a Java-based implementation of the FHIR specification, licensed under Apache 2.0.

Feedback and Support: We value your feedback and suggestions. If you have any thoughts, suggestions, or issues to report, please contact us at fhir@whitefox.cloud.

Further Information: For a comprehensive introduction to this server as a reference implementation, please refer to the article on Whitefox.cloud website.

Source Code: The source code for this server is available on the Whitefox GitHub repository. You can use it locally or customize it to meet your specific needs. Visit our GitHub repository.
